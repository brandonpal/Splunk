This visualization reads these fields from your search results (one row per panel):

metricValue — compliance % (0–100), drives the bar width and "(P%)" text.
threshold — red, yellow, or green. Drives the bar and pill color.
compliant — compliant asset count, shown as the first number in "N of M".
noncompliant — non-compliant asset count. Combined with compliant to get the "of M" total.

Row labels (e.g. "BBC (On-Prem)") are not pulled from the data — set them via the panel title, since each platform/row is its own panel using this same visualization.

Example search:
| stats latest(metricValue) as metricValue latest(threshold) as threshold latest(compliant) as compliant latest(noncompliant) as noncompliant
